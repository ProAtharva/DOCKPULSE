DockPulse

Real-time port traffic simulator · personal project

Fake ships, berths, and container counts advance on a 2 second tick. The UI streams updates over WebSockets — no refresh, no button clicks. Stack: Node.js, Express, Socket.io, Redis Pub/Sub · decision engine assigns berths and scores congestion.
